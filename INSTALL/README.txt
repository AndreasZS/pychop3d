'setup_linux' is a Bash script 
It is designed to install the packages needed to run pychop3d
It can be used in Linux or Windows Subsystem for Linux
It makes use of 'Apt' and will NOT work in Cygwin
It setup and activate a python virtual environment in your pychop3d directory
To do this:
1. Navigate to the main pychop3d directory
2. Run the script with either:
   source INSTALL/setup_linux
   or
   . INSTALL/setup_linux

note: 'source' or '.' are needed to create and activate a virtual environment in your current shell
